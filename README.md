# 30天英语口语+词汇突破计划

> 制定日期：2026年3月25日
> 目标人群：CET-6基础，词汇量下降，会读不会说
> 学习目标：词汇量回升1000+，敢开口，能进行日常对话

---

## 一、学习者画像

| 维度 | 现状 | 目标 |
|------|------|------|
| **词汇量** | 约3000-4000（下降） | 回升1000+ |
| **发音** | 会读，发音OK | 继续保持 |
| **口语** | 不会说、不敢说 | 能进行日常对话 |
| **问题** | 词汇遗忘+不会输出 | 恢复词汇量+形成口语习惯 |

---

## 二、两大核心问题

| 问题 | 解决方案 |
|------|---------|
| **词汇量下降** | 每天背单词+语境记忆+主动使用 |
| **会读不会说** | 大量输出练习+场景化学习 |

---

## 三、学习原则

1. **词汇+口语结合**：背的词当天就要用
2. **先说对，再说好**：先能表达，再追求地道
3. **大声说出来**：不说等于没学
4. **坚持打卡**：每天学习比突击有效

---

## 四、每日学习结构（60分钟）

| 时间 | 内容 | 说明 |
|------|------|------|
| **5分钟** | 复习昨天词汇 | 巩固记忆 |

| **15分钟** | 口语学习 | 句型/场景 |
| **15分钟** | 输出练习 | 用新词造句+口语练习 |
| **10分钟** | 录音复盘 | 听自己的问题 |

### 5.1 词汇范围

按场景和频率学习，而非按字母表：

| 阶段 | 词汇范围 | 数量 |
|------|---------|------|
| 第1周 | 基础动词+日常名词 | 200词 |
| 第2周 | 生活场景词汇 | 200词 |
| 第3周 | 工作职场词汇 | 200词 |
| 第4周 | 表达观点词汇 | 200词 |

### 5.2 词汇记忆方法

| 方法 | 说明 |
|------|------|
| **语境记忆** | 在句子中记单词 |
| **主动造句** | 每个词造1个句子 |
| **分类记忆** | 按主题分类学习 |
| **间隔复习** | 第1/3/7天复习 |

### 5.3 词汇工具

- **墨墨背单词**：间隔复习算法
- **百词斩**：图片记忆
- **欧路词典**：查词+生词本
