## 二、核心问题分析

**为什么"会读不会说"？**

| 原因 | 说明 |
|------|------|
| **被动词汇多** | 认识的词 ≠ 能主动用的词 |
| **缺少输出练习** | 光看不练，没有形成肌肉记忆 |
| **害怕犯错** | 心理障碍，不敢开口 |
| **没有口语思维** | 想到中文再翻译，反应慢 |

**解决方案：**
- 把被动词汇变成主动词汇
- 大量输出练习，形成肌肉记忆
- 从简单句开始，建立自信
- 培养英语思维，减少翻译过程

---

## 三、学习原则

1. **先说对，再说好**：先能表达，再追求地道
2. **大声说出来**：不说等于没学
3. **场景化练习**：学完就能用
4. **重复是关键**：同一句话要说10遍以上

## 七、学习方法

### 核心方法：影子跟读法（Shadowing）

1. 选择一段音频（1-2分钟）
2. 听一遍，理解内容
3. 跟读，模仿语调
4. 看着文本读
5. 脱稿复述

### 口语输出方法

| 方法 | 说明 |
|------|------|
| **自言自语** | 看到什么说什么，养成开口习惯 |
| **录音回听** | 听自己的问题，针对性改进 |
| **自问自答** | 模拟对话场景 |
| **复述练习** | 看完内容后用自己的话说出来 |
