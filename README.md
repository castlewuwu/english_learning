## 二、词汇学习策略

### 为什么CET-6词汇量还下降？

| 原因 | 解决方案 |
|------|---------|
| 被动词汇多、主动词汇少 | 学搭配、造句、输出 |
| 长期不用遗忘 | 复习+实际使用 |
| 只记意思不会用 | 场景化学习 |

### 词汇选择原则

不学太简单的词，而是学：
- **易忘的高频词**：认识但说不出来的词
- **口语常用词**：日常生活、工作中高频使用
- **表达升级词**：让口语更地道的词
